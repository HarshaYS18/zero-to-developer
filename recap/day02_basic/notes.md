## 🔹 Problems I Solved

### 1. Palindrome Number (without string conversion)
- Used % to get last digit
- Used // to remove last digit
- Built reverse number step-by-step

👉 Key learning:
- % gives last digit
- // removes last digit
- Loop continues until number becomes 0

---

### 2. Reverse a Number
- Used same logic as palindrome
- Stored reversed number in variable

👉 Key learning:
- Logic reuse is powerful
- Same pattern applies to multiple problems

---

### 3. Armstrong Number
- Extracted each digit using %
- Raised digit to power (number of digits)
- Added all values and compared with original

👉 Key learning:
- Can break number into digits using loops
- Learned how to apply math logic inside loop

---

### 4. Pattern Printing (Nested Loops)
- Used outer loop for rows
- Used inner loop for columns
- Printed numbers step-by-step

👉 Key learning:
- Nested loops control structure of patterns
- Outer loop = rows
- Inner loop = content inside each row

---

## 🔹 Important Concepts I Understood

- while loop → used when condition-based repetition
- for loop → used when range is known
- range(start, end) → end is excluded
- Nested loops = loop inside another loop
