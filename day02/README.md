# Day 2 – Python Loops 🔁

**Author:** Sree Harsha Reddy

This day was focused on mastering loops through problem-solving. I practiced various loop-based problems involving numbers, strings, and lists.

---

## 📁 Problems Solved

1. Print numbers from 1 to 50  
2. Print even numbers from 1 to 50  
3. Find sum of numbers from 1 to 100  
4. Multiplication table (user input)  
5. Count vowels in a string  
6. Reverse a string (without slicing)  
7. Find largest number in a list (user input)  
8. Remove duplicates from a list (without using set)  
9. Pattern printing using nested loops  

---

## 💡 Concepts Learned

- for loops  
- range(start, stop, step)  
- while loops (basics)  
- Nested loops  
- String iteration  
- List traversal  
- Conditional logic inside loops  
- Input handling and type conversion  

---

## 🚀 Key Takeaways

- Loops are essential for automating repetitive tasks  
- Problem-solving improves with practice, not theory  
- Writing logic manually builds strong fundamentals
